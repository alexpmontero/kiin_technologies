CÁLCULO INTEGRAL DE RESERVAS Y MAPEO

ACI: Adams Consulting International

www.acibarbados.com

- Solución al “Garbage IN / Garbage OUT”.
Este software permite integrar en una sola base de datos la información que viene de otros sistemas como Halliburton (Decision Space, Seisworks, OpenWorks), Schlumberger (Petrel, OFM), Paradigm (Skua, etc), Kingdom y otros, y realizar correlaciones entre bases de datos y múltiples chequeos (cross-checks) que permiten consolidar y validar toda la información disponible, reforzando así la calidad y exactitud de los modelados a realizar.
+ Cálculos de reservas petroleras.
Cálculos de reservas y validación de localizaciones, y posiciones de nuevos pozos con respecto a trampas estructurales y estratigráficas. Validación de pozos de relleno (in-fill drilling) y pozos estratigráficos.
+ Consolidación de base de datos.
Consolidación de base de datos en un solo sistema y eliminación de datos incoherentes y redundantes.
+ Generación de mapas de exploración.
Generación de mapas exhaustivos geológicos & geofísicos para exploración y producción incorporando datos de pozos, de horizontes sísmicos, de topografías (ArcGis), datos de shapefiles, datos de ingeniería y producción, datos geomecánicos y de riesgo CRS, entre otras.
+ Validación.
A nivel de producción, la ventaja es de poder validar que toda la información que sirve para modelar el comportamiento del yacimiento que está válida y que el volumen de reserva calculado de manera estática es coherente con lo que se observa a nivel de producción.
